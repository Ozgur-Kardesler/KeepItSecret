KeepItSecret is a lightweight NeoForge server sided mod that allows partial occlusion in nametags. There are two gamerules provided for control over player and mob nametags, you can turn on and off this feature by these gamerules to your liking.
You can find us on [Modrinth](https://modrinth.com/mod/keepitsecret) and [Curseforge](https://www.curseforge.com/minecraft/mc-mods/keepitsecret)!
